Curricular

Projects which formed a part of my graduation requirements or were part of work done as part of internships.

  2. Color Document Image Binarization: Converted a given color document image to a binary image while retaining all desired image elements.
  3. Dual Channel Electroencephalography Headset: Implemented a dual channel electroencephalography headset.
